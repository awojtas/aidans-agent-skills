# 08 — Open Questions

The **waiting room** (Volere's term). Anything not ready to become a requirement statement — but worth not forgetting — lives here. Each entry has a question, an owner, and a next step.

## How to use this file

- A user hesitates or says "I'm not sure" → add an entry.
- An agent has to make a default choice without confirmation → add an entry.
- A requirement is marked Draft because of a missing fit criterion → add an entry.
- A "Not yet" non-goal from `01-goals-and-non-goals.md` → add an entry with a planned revisit date.
